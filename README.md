## Чтобы начать

```sh
  Открываете папку backend. Меняете данные в backend/index.js pool на данные своей postgresql БД.
```

```sh
  Устанавливаете зависимости:
  yarn
```

```sh
  Запускаете сервер:
  node index.js
```

```sh
  Открываете папку frontend.
```

```sh
   Устанавливаете зависимости:
  yarn
```

```sh
  Запускаете:
  yarn start
```

```sh
  Наслаждаетесь
```
